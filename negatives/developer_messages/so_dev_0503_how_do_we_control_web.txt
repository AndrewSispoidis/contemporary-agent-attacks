How do we control web page caching, across all browsers?. Our investigations have shown us that not all browsers respect the HTTP cache directives in a uniform manner. For security reasons we do not want certain pages in our application to be cached, ever, by the web browser. This must work for at least the following browsers: Internet Explorer 6+ Firefox 1.5+ Safari 3+ Opera 9+ Chrome Our requirement came from a security test. After logging out from our website you could press the back button and view cached pages.
